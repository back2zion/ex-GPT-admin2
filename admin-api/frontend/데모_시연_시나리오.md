# 도로공사 사용자 관리 시스템 개선 데모 시나리오

## 📋 개요
이 문서는 도로공사 직원에게 개선된 사용자 관리 시스템을 시연하기 위한 시나리오입니다.

## 🎯 데모 목적
기존 시스템에서 불편했던 "한 명씩 권한을 부여해야 하는 문제"를 해결하고, 대량 권한 관리 기능을 시연합니다.

---

## 📊 데모 데이터 구성

### 실제 도로공사 조직 구조 반영
- **본부**: 서울경기본부, AI디지털본부, 홍보처, 건설본부, 기술본부, 영업본부, 강원본부, 충청본부, 전라본부, 경상본부
- **부/처**: 기술처, 디지털관리처, 홍보계획부, 언론홍보부, 시설관리처, 건설관리처, 안전관리처, 기획처, 재무처, 인사처
- **팀**: 시설부, 경영시스템부, 홍보계획 총괄, 디지털 소통, 건설기획부, 도로유지부, 시설안전부, 전기통신부, 교통관리부, 재무관리부
- **직종**: 전기, 정보통신, 토목, 건축, 기계, 사무직, 기술직, 안전관리, 경영지원
- **직급**: 3급, 4급, 5급, 6급, 7급, 8급
- **직위**: 본부장, 처장, 차장, 부장, 대리, 주임, 사원
- **권한 비율**: 약 80%가 권한 보유, 20%가 미보유
- **미접속 기간**: 일부 사용자는 90일 이상 미접속

## 🎬 시연 시나리오

### 1단계: 기존 문제점 시연 (Before) - 30초
**시나리오**: "신규 입사자 20명에게 GPT 권한을 부여해야 하는 상황"

1. 사용자 관리 탭 열기
2. 권한이 없는 사용자(권한 = N) 확인
3. **문제점 강조**: "기존에는 이 20명을 일일이 선택해서 권한을 부여해야 했습니다"

```
❌ 기존 방식: 20명 × 3클릭 = 60번 클릭 필요
   - 사용자 선택 (1클릭)
   - 권한 부여 버튼 (1클릭)
   - 확인 (1클릭)
   → 20번 반복
```

---

### 2단계: 빠른 필터 기능 시연 (Improved) - 30초
**개선 기능**: 빠른 필터 버튼으로 특정 조건 사용자를 한 번에 필터링

1. **"권한 없는 사용자" 버튼 클릭**
   - 화면 상단의 파란색 "권한 없는 사용자" 버튼 클릭
   - 권한이 없는 사용자만 즉시 필터링됨
   - 리스트가 자동으로 업데이트됨

2. **"장기 미접속자 (90일+)" 버튼 클릭**
   - 90일 이상 미접속한 사용자만 표시
   - 권한 회수가 필요한 사용자를 쉽게 찾을 수 있음

```
✅ 개선: 원하는 사용자 그룹을 1클릭으로 필터링
```

---

### 3단계: 대량 선택 기능 시연 (Improved) - 30초
**개선 기능**: 필터링된 사용자를 한 번에 선택

1. 권한 없는 사용자 필터 적용 상태에서
2. **"현재 페이지 모두 선택" 체크박스 클릭**
3. 파란색 요약 패널 표시 확인:
   ```
   📊 20명 선택됨
   [선택된 20명에게 권한 부여] [선택된 20명의 권한 회수] [선택 해제]
   ```

```
✅ 개선: 20명을 1클릭으로 선택
```

---

### 4단계: 대량 권한 부여 시연 (Improved) - 1분
**개선 기능**: 선택된 사용자에게 한 번에 권한 부여

1. **"선택된 20명에게 권한 부여" 버튼 클릭**
2. 모델 선택 다이얼로그 표시:
   - 70B
   - Qwen235B
   - Qwen32B
3. **"70B" 선택 후 확인**
4. **진행 상황 모달 표시**:
   ```
   ⏳ 대량 작업 진행 중

   [회전하는 원형 프로그레스]

   20명에게 권한을 부여하는 중...

   [━━━━━━━━━━━━━━━━━━] 75%

   75% 완료
   ```
5. 완료 메시지 표시: "20명에게 권한이 부여되었습니다"
6. 리스트 자동 새로고침으로 변경 사항 확인

```
✅ 개선 결과:
   - 기존: 60번 클릭
   - 개선: 4번 클릭 (필터 → 전체선택 → 권한부여 → 모델선택)
   - 시간 절약: 약 93% 감소
```

---

### 5단계: 검색 및 정렬 기능 시연 (Bonus) - 30초
**추가 개선 기능**

1. **검색 기능**:
   - 검색 타입 선택: 전체 / 사번 / 이름 / 본부 / 부처 / 팀
   - "서울경기본부" 검색
   - 해당하는 사용자만 표시

2. **정렬 기능**:
   - 컬럼 헤더 클릭으로 정렬
   - 최종접속일 ↓ 클릭 → 오래된 순 정렬
   - 직급 클릭 → 3급부터 순서대로 정렬

3. **페이지네이션**:
   - 10/20/30/40/50개씩 보기 선택
   - 페이지 이동 버튼

```
✅ 추가 편의 기능: 유연한 검색, 정렬, 페이지 설정
```

---

### 6단계: 통계 및 엑셀 다운로드 (Bonus) - 20초

1. **통계 버튼 클릭**:
   ```
   📊 사용자 통계

   총 사용자: 200명
   권한 보유: 160명 (80%)
   권한 미보유: 40명 (20%)

   모델별 사용자:
   - 70B: 54명
   - Qwen235B: 53명
   - Qwen32B: 53명
   ```

2. **엑셀 다운로드**:
   - 현재 필터링된 데이터를 엑셀로 다운로드
   - 파일명: `users_YYYYMMDD_HHMMSS.xlsx`

## 🎥 동영상 촬영 팁

### 화면 설정
- 브라우저 주소: `https://ui.datastreams.co.kr:20443/admin/#/users`
- 브라우저 확대: 100% (또는 90%로 더 많이 보이게)
- 화면 해상도: 1920x1080 권장

### 녹화 도구
- **Linux**: SimpleScreenRecorder, OBS Studio
  ```bash
  # SimpleScreenRecorder 설치
  sudo yum install epel-release
  sudo yum install simplescreenrecorder

  # 또는 OBS Studio
  flatpak install flathub com.obsproject.Studio
  ```
- **Windows**: OBS Studio, Bandicam
- **Mac**: QuickTime Player, OBS Studio

### 편집 포인트
1. **Before 시연**: 기존 방식의 불편함 강조
2. **After 시연**: 새로운 기능의 편리함 강조
3. **시간 비교**: "60번 클릭 → 4번 클릭"을 자막으로 표시
4. **진행 상황**: 모달의 프로그레스 바를 명확히 보여주기
