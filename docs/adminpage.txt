1. 로그인
• 로그인 화면
• 관리도구 로그인 화면입니다.(관리도구는 허가된 사용자만 접속할 수있습니다)
• 관리도구 URL에 접속합니다.(외부에서는 접속할 수 없습니다)
1. 최초 부여받은 아이디를 입력합니다
2. 비밀번호를 입력 후 로그인 버튼을 클릭합니다
3. 아이디 기억하기를 체크하면 이후 로그인 시 아이디가 입력되어 표출됩니다.
* 비밀번호는 정보변경을 통해 추후 변경할수 있습니다

2. 통계

1) ex-GPT 통계
• 최근의 통계내역을 확인합니다
1. 특정 기간을 설정하면 해당 기간 동안의 통계 내용을 확인할수 있습니다
2. 초기화 버튼을 클릭하면 검색기간이 처음 진입 시 날짜로 변경됩니다
3. 엑셀 다운로드 아이콘을 클릭하면 해당 데이터를 엑셀로 다운로드할 수 있습니다
4. 메뉴 체크에 따라 그래프에 해당 메뉴의 방문 건수가 노출되거나 숨겨집니다

1. 기간내 총질문수와 메뉴별 질문수를 확인합니다. 메뉴클릭시 해당메뉴의 통계를 제외한 나머지를 확인할수 있습니다
2. 기간내 메뉴별 총 오류신고수를 확인합니다. 메뉴클릭시 해당메뉴의 통계를 제외한 나머지를 확인할수 있습니다
3. 항목 선택에 따라 일자별로 질문수와 오류신고수를 그래프로 확인합니다
4. 엑셀 다운로드 아이콘을 클릭하면 해당 데이터를 엑셀로 다운로드할 수 있습니다

1. 대상 문서관리에 등록된 카테고리별 문서 등록 현황을 확인 합니다
2. 질문 내용을 카테고리별로 분류한 전체 건수와 통계 내역을 그래프로 확인합니다
3. 엑셀 다운로드 아이콘을 클릭하면 해당 데이터를 엑셀로 다운로드할 수 있습니다

2) 서버현황 통계
• 서버현황을 확인합니다
1. 조회일 이전일자에 대한 시간대별 검색건수를 확인합니다
2. 더보기 클릭 시 상세 검색 통계 화면으로 이동합니다
3. 더보기 클릭 시 상세 프롬프트 통계 화면으로 이동합니다

3. 대화내역 조회
• 대화내역 목록
• 사용자들이 ex-GPT와 주고받은 내용들을 조회하고 확인합니다
1. 기간을 별도로 설정하여 검색할 수 있습니다. 
2. 상세내역을 확인하려면 질문이나 답변을 클릭합니다
3. 한 페이지에 보이는 목록의 개수를 지정할 수 있습니다. 숫자지정 시 바로 반영됩니다.
4. 현재 보이는 페이지를 엑셀로 다운로드합니다
5. 슈퍼관리자 로그인 시만 보이는 다운로드 버튼입니다. 화면에 보이는 항목 외에 사번, 성명 등이 포함된 파일을 다운로드할수 있습니다.

• 대화내역 상세
• 선택한 대화에 대해 상세 내용을 확인합니다
• 대화 메뉴에 따라 상세화면은 다르게 표현됩니다
1. 사용자 정보, 질문시간,질문내용 및 세션 ID등 기타 내역들이 표출됩니다
2. 질문과 답변 내역 일부가 표시됩니다.
3.  전체 보기 클릭 시 팝업으로 전체 내용을 확인할 수 있습니다
4. 목록 버튼 클릭 시 목록 화면으로 이동합니다

4. 권한관리
1) ex-GPT 접근권한>사용자관리
• ex-GPT를 사용할 수 있는 사용자를 지정하거나 장기 미접속자에 대한 차단을 설정합니다
1.  30일 이상 접속하지 않은 사용자를 검색하려면 장기 미접속자를 체크합니다
2.  사용자를 선택 후 권한 회수 버튼을 클릭하면 선택 사용자에한 해 GPT 접근을 허용하지 않습니다
3. 선택 사용자에 대해 GPT 접근을 일괄적으로 허용합니다. 적용할  모델을 선택합니다
4. 개별로 권한을 주거나 회수할수 있으며 사용할 모델을 지정할 수 있습니다. 변경 시 즉시 적용됩니다
5.  부서별 사용자 통계가 팝업으로 출력됩니다

1) ex-GPT 접근권한>접근승인관리
• ex-GPT 사용 신청자 현황을 확인할 수 있으며 신청을 승인 처리합니다
1. 상태 값을 신청, 미신청, 거부로 검색하여 명단을 확인할 수 있습니다
2. 여러 명을 선택하여 사용자에대해 일괄적으로 사용을 승인처리합니다
3. 개별적으로 사용을 승인 처리합니다

2) 국정자료 권한관리
• 직원을 검색하여 문서 권한을 추가 할 수 있습니다
1. 팀명, 이름, 사번 등으로 직원을 검색 후 문서 권한을 추가합니다.
2. 한 페이지에 보이는 목록의 개수를 지정할 수 있습니다.
3.  문서 권한을 부여할 사용자를 선택하면 문서 권한 추가 페이지로 이동합니다.

2) 국정자료 권한관리>문서권한 추가
• 선택한 직원에게 문서 권한을 추가 합니다
1. 선택한 직원의 기본 정보가 출력됩니다.
2.  문서 권한을 부여할 부서를 선택합니다. 선택을 마치면 추가버튼을 클릭합니다.
3. 권한을 부여받은 사용자는 국회자료 검색 시 선택한 부서에서 만든 자료를 열람할 수 있습니다.
4. 부서 삭제 시 해당 부서에서 작성한 문서는 열람할 수 없습니다. 또한 본인이 속해있는 부서는 삭제할수 없습니다.
5. 클릭 시 문서 권한 목록 페이지로 이동합니다.




5. 학습데이터 관리
1) 대상문서 관리>카테고리 조회
• 대상 문서를 등록하고 조회합니다.
카테고리별로 등록된 문서수가 보입니다
1. 카테고리명 선택 시 해당 문서가 목록에 출력됩니다
2. 신규 카테고리를 생성하려면 카테고리 생성 버튼을 클릭합니다
3. 새로운 문서를 등록하려면 문서등록 버튼을 클릭합니다
4. 문서를 삭제하려면 삭제할 문서를 먼저 선택 후 삭제 버튼을 클릭하면 삭제됩니다
5. 파일명을 클릭하면 선택한 문서의 전 처리 분류 목록 페이지로 이동합니다

1) 대상문서 관리>카테고리 생성
• 카테고리를 생성합니다
1. 카테고리 생성 버튼을 클릭하여 카테고리 생성 팝업을 출력합니다
2. 카테고리명을 필수로 입력합니다. (이미 사용 중인 카테고리명은 사용할 수 없습니다)
3. 패시징 패턴을 입력 후 타이틀 형식을 지정합니다.
4. 추가 버튼을 클릭하여 추가 입력할 수 있습니다

1) 대상문서 관리>대상문서 업로드
• 대상 문서를 특정 경로에 업로드합니다
1. 카테고리를 먼저 선택 후 패시징 방식을 선택합니다. 토큰 선택 시 최대 글자 수를 입력합니다
2. 경로를 차례대로 선택합니다.
3.  파일을 업로드합니다. 파일을 업로드하면 파일명이 출력되며 삭제 아이콘 클릭 시 업로드 파일이 삭제됩니다
4. 찾는 폴더가 없을 경우 폴더 생성 버튼을 클릭하여 새로운 폴더를 생성 후 업로드할 수 있습니다.

1) 대상문서 관리>전처리분류 목록확인
• 선택한 문서의 전처리 분류 항목을 확인하고 분류 내용을 추가로 등록 할  수 있습니다.
1. 등록 및 수정 버튼을 클릭하면 전처리 분류 신규 등록 화면으로 이동합니다. 등록한 내용이 현재 페이지에 표시됩니다.
2. 목록을 삭제하려면 삭제할 목록을 먼저 선택 후 삭제 버튼을 클릭하면 삭제됩니다.
3. 문서 목록 버튼을 클릭하면 대상 문서 관리목록 화면으로 이동합니다.

1) 대상문서 관리>전처리분류 등록
• 선택한 문서의 전처리분류 항목을 등록합니다.
복사 내용 선택
1. 선택한 문서의 기본등록정보를 보여집니다
2. 선택한 문서의 본문내용이 출력 됩니다.
3. 필요한 영역을 마우스로 선택후 상단의 복사버튼을 클릭합니다.(복사내용이 클립보드에 저장되었습니다)
4. 우측영역에서 붙여넣을 영역을 선택후 붙여넣기 버튼을 클릭합니다. 복사된 텍스트가 해당영역에 삽입됩니다.
5. 등록버튼을 클릭하여 작성내용을 저장합니다. 목록버튼을 클릭하기 전까지 현재페이지에서 계속 등록할수 있습니다.

2) 사전 관리>사전목록
• 사전 목록을 확인 하고 새로운 사전을 등록합니다
1. 사전명 또는 설명으로 등록된 사전을 검색할수 있습니다.
2. 사전명을 클릭하면 사용자 사전목록페이지로 이동합니다.
3.  새로운 사전을 추가하려면 사전 추가버튼을 클릭하여 사전추가 팝업을 출력합니다.
4. 사전내용을 수정하려면 수정아이콘을 클릭하여 사전을 수정할 수 있습니다.
5. 등록된 사전을 삭제하려면 사전 종류를 선택후 삭제버튼을 클릭합니다.
6. 개별적으로 삭제 아이콘을 클릭하여 삭제할수 있습니다

2) 사전 관리>사전추가
• 새로운 사전을 추가합니다.
1. 사전추가버튼을 클릭하여 사전추가 팝업을 호출합니다.
2. 사전종류를 선택한후 사전명을 입력합니다. 사전종류, 사전명은 필수 항목입니다
3. 사전종류가 동의어 사전, 금칙어 사전일 경우 하단에 대소문자 구분 체크박스가 표시됩니다

2) 사전 관리>사전동기화
• 관리도구 사전을 엔진내부에서 가져오거나 엔진내부의 사전을 관리도구에 동기화 합니다 .
1. 가져오기 클릭시 엔진내부의 데이터를 관리도구 데이터로 다운로드 합니다
2.  동기화 클릭시 관리도구의 데이터를 엔진에 업로드 합니다

2) 사전 관리>사용자 사전 추가
• 사용자사전 목록을 확인하고 새로운 어휘를 추가/삭제합니다
1. 새로운 어휘를 추가하려면 어휘 추가 버튼을 클릭하여 어휘추가하기 팝업을 출력합니다. 검색어나 분리정보 입력은 필수입니다.
2. 등록된 어휘를 삭제하려면 먼저 검색어를 선택후 삭제버튼을 클릭합니다. 또는 개별적으로 삭제 아이콘을 클릭하여 삭제할수 있습니다
3. 어휘내용을 수정하려면 수정아이콘을 클릭하여 수정할수 있습니다. 검색어는 수정할수 없습니다

2) 사전 관리>사용자사전 업로드 및 다운로드
• 사용자사전을 업로드 하거나 다운로드 합니다
1. 업로드 또는 다운로드를 선택합니다
2. 업로드 클릭시 업로드 옵션을 선택한 후 업로드하려는 파일을 첨부합니다. 업로드버튼을 클릭하면 사용자 사전이 변경됩니다.
3. 도움말을 클릭하면 샘플파일을 다운로드 받을수 있습니다
4. 다운로드 클릭시 현재 사용자 사전을 다운로드 합니다



6. 부가서비스 관리
1) 인사말 관리
• ex-GPT 각 메뉴진입시 보여지는 인사말을 확인 및 등록합니다
1. 수정버튼 클릭시 인사말 등록페이지로 이동합니다.

1) 인사말 관리>인사말 수정
• 인사말을 등록 및 수정합니다
1. 등록할 내용을 입력하거나 수정합니다.
2. 내용이 저장되며 목록페이지로 이동됩니다. 저장과 동시에 ex-GPT 화면에 바로 적용됩니다.
3. 작성내용을 취소하고 목록페이지로 이동합니다
2) 공지사항>공지사항 목록

• 공지사항 목록내용을 확인 하고 새로운 공지사항을 등록합니다
1. 제목 또는 내용으로 공지사항을 검색할수 있습니다.
2.  등록버튼을 클릭하면 공지사항 등록페이지로 이동합니다.
3. 다운로드버튼을 클릭하면 공지사항 내용을 엑셀파일로 내려받을수 있습니다
4. 제목을 클릭하면 공지사항 상세페이지가 출력됩니다.

2) 공지사항>공지사항 상세
• 공지사항 상세내용을 확인하거나 삭제합니다

1. 공지사항 상세내역을 확인할수 있습니다. 상세페이지에서는 수정을 할수 없습니다.
2. 첨부파일 클릭시 파일을 다운로드 합니다
3. 삭제버튼 클릭시 해당 공지사항이 삭제됩니다.
4. 수정버튼 클릭시 수정페이지로 이동합니다.

5. 목록버튼 클릭시 이전 목록페이지로 이동합니다

2) 공지사항>공지사항 수정
• 공지사항 내용을 수정합니다
